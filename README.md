# Finance Tracker API

A personal finance tracking backend built with **ASP.NET Core**, **Entity Framework**, and **SQL**.

## Features

- Track accounts, income, and expenses
- Categorize transactions
- Monthly spending analytics
- REST API built with ASP.NET Core
- Relational database with EF Core

## Tech Stack

- C#
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- SQL

## Running Locally

```bash
dotnet run
